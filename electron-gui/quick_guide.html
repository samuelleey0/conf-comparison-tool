Quick Guide

Follow these simple steps to get started with the Automated Cisco Configuration & Marking System.

1

Device Setup

Prepare your Cisco device and establish a baseline configuration.

  • Connect via Serial or SSH
  • Verify device responsiveness
  • Create baseline snapshot
  • Confirm setup complete
2

Create Directory

Set up the workspace for storing student files and session data.

  • Enter exam name
  • Specify tutor/classroom
  • Set time slot
  • Add student ID
3

Run Connection

Execute the grading workflow and collect student configuration data.

  • Select connection method
  • Configure parameters
  • Start execution
  • Monitor progress
4

Review Results

Examine marks, collected data, and execution logs.

  • View overall marks
  • Review config files
  • Check execution logs
  • Download results
📁 Data Storage
All student data and results are stored in D:\Documents organized by exam, tutor, time slot, and student ID.

Helpful Tips

🔌 Connection Stability

Do not disconnect the device while execution is running. Connection interruption may result in incomplete data collection.

⚙️ Device Preparation

Always verify baseline configuration before proceeding. A proper baseline ensures accurate grading results.

✓ Data Verification

Review results immediately after completion to ensure all data was collected successfully before proceeding.

📊 Additional Tools

Use System Admin for configuration and Sample Logs for validation and troubleshooting issues.

For more help or issues, contact your system administrator.
