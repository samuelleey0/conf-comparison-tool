Turn on when the first row is only a report title, for example Active Student Listing. That row will be skipped.

Turn on when the next row contains column labels, for example Student Number and Student Name. That row will be skipped.

Turn on when the first column is only row numbering such as 1, 2, 3. Student ID will be read from the next column.


These switches are on by default for exported student-list files that include a title row, column labels, and row numbers. If your file already starts with student data, turn these switches off. With all switches off, the importer reads column 1 as Student ID and column 2 as Student Name.
