Quick Guide

How to use the app

Follow the workflow from left to right when setting up a session. The homepage links below map directly to each stage, so staff can move through the process in a predictable order.

1 Set up the device
2 Choose the directory
3 Run the connection
4 Review the results
5 Export or manage support
Step 1 Open Device Setup to prepare the hardware and baseline exam environment.
Step 2 Use Directory to choose or create the working folder for the current student or session.
Step 3 Go to Connection to start the active grading, execution, or device communication process.
Step 4 Open Results to inspect marks, generated outputs, and the completion state for the session.
Step 5 Use Send To Melbourne, Sample Logs, or System Admin for export, log collection, and maintenance tasks.
